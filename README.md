## Data 
Vehicle data is classified into 16 distinct columns: 
* [0] City 
* [1] Price 
* [2] Year
* [3] Make
* [4] Model 
* [5] Condition
* [6] Cylinders 
* [7] Gas type 
* [8] Odometer
* [9] Title status 
* [10] Transmission type
* [11] Drive type 
* [12] Size 
* [13] Car type 
* [14] Color
* [15] State
